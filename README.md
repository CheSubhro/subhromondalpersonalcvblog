# SubhroMondalPersonalCVBlog

A modern personal portfolio and blogging platform built with Next.js, React, Tailwind CSS, and MongoDB.

## Features

- Personal portfolio and CV section
- Dynamic blog system
- Responsive modern UI
- Fast performance with Next.js
- MongoDB integration using Mongoose
- Reusable React components
- Mobile-friendly design

## Tech Stack

- Next.js 14
- React 18
- Tailwind CSS
- MongoDB
- Mongoose
- React Icons

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/subhromondalpersonalcvblog.git

Go to the project directory:

cd subhromondalpersonalcvblog

Install dependencies:

npm install
Run Development Server
npm run dev

Open:

http://localhost:3000
Build for Production
npm run build
npm start
Environment Variables

Create a .env.local file and add:

MONGODB_URI=your_mongodb_connection_string
